## How to run it

#### Requirements
You should to have a Node.js environment on your computer

#### Run

* clone this repo
```terminal
git clone git@github.com:berbecki/PromoSE.git
```

* install dependencies
```terminal
npm install
```

* run app
```terminal
npm install
```

* open the app in your browser
http://localhost:8080
